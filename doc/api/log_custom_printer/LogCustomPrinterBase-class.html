LogCustomPrinterBase class Core - Sistema de Logging

Classe principal do sistema de logging.

Implementa o padrão Singleton para centralizar a configuração do sistema de logging. Permite escolher entre diferentes impressoras de log (LogSimplePrint ou LogWithColorPrint) e gerenciar a configuração global.

Exemplo básico:

// Configuração simples (padrão)
final printer = LogCustomPrinterBase();

// Configuração com cores
final colorPrinter = LogCustomPrinterBase.colorPrint();

// Configuração customizada
final customPrinter = LogCustomPrinterBase(
  logPrinterCustom: LogWithColorPrint(
    config: ConfigLog(
      enableLog: true,
      onlyClasses: {DebugLog, ErrorLog},
    ),
  ),
);
