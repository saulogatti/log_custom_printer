LoggerPersistenceService
description

LoggerPersistenceService class final Utilitários

Serviço de persistência e consulta de logs.

Encapsula operações de armazenamento, recuperação, limpeza e busca sobre o repositório de logs configurado.

Quando logOutputHandler está definido, operações que alteram o estado do cache notificam o consumidor com a lista de logs atualizada.
