DateTimeLogHelper
description

DateTimeLogHelper extension Utilitários

Extension para formatação de data/hora em logs.

Fornece métodos utilitários para formatar timestamps de forma consistente nos logs gerados pela biblioteca.

Exemplo de uso:

final agora = DateTime.now();
print(agora.onlyTime());      // "14:30:25.123"
print(agora.onlyDate());      // "18/11/2025"
print(agora.logFullDateTime); // "18/11/2025 14:30:25.123"
