ILoggerCacheRepository
description

ILoggerCacheRepository class abstract interface Core - Sistema de Logging

Contrato para persistência e consulta de logs.

Define as operações mínimas para armazenar, recuperar e limpar objetos de log no cache da biblioteca.

Implementações podem usar apenas memória, arquivo local, banco de dados ou qualquer outro backend, desde que respeitem as assinaturas e o comportamento assíncrono descrito aqui.
