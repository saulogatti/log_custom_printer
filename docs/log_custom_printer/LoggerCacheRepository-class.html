LoggerCacheRepository class abstract interface Utilitários

Interface para repositório de cache de logs.

Define as operações básicas para armazenamento, recuperação e limpeza de logs. Caso queira personalizar o armazenamento dos logs (ex: usar um banco de dados local, SharedPreferences, etc.), implemente esta interface e forneça a implementação personalizada para o LogPrinterService via registerLogPrinter.
